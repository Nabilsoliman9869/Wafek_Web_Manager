# تحليل سند قبض.xml واستخدامه في ميل الموافقة

## الحقول المستخرجة من نموذج سند قبض (FrmBond)

| FieldSource (في التليورك) | التسمية في النموذج | الاستخدام في الميل |
|--------------------------|---------------------|-------------------|
| **NmbBondNumber** | رقم | رقم السند ✓ (موجود) |
| **InDate** | التاريخ | تاريخ السند |
| **xtrSearchData3** | الفرع | اسم الفرع |
| **SrhCostCenter** | مركز الكلفة | مركز الكلفة |
| **xtrSearchData1** | المشروع | اسم المشروع |
| **SrhCurrency** | العملة | العملة |
| **SrhAccount** | الحساب | الحساب (للسندات ذات حساب واحد) |
| **xtrDataText1** | ملاحظات | الملاحظات ✓ (موجود) |
| **NmbCredit** | المجموع | المبلغ الإجمالي (دائن) |

## ربط الحقول بجداول قاعدة البيانات (من Fix_Command_9009 والكود)

| الحقل | الجدول المرجعي | العمود/العلاقة |
|-------|----------------|-----------------|
| الفرع | TBL050 | BronchName (لاحظ: Bronch وليس Branch) |
| مركز الكلفة | TBL005 | CostCenter أو CostCenterName |
| المشروع | TBL049 | ProjectName |
| العملة | TBL001 | CurrencyName |
| الحساب | TBL004 | AccountName |
| التفاصيل/المجموع | TBL011+TBL012 | SUM(Credit) أو من TBL011 |

## الجداول المستخدمة

- **TBL010** أو **TBL011**: رأس السند (CardGuide)
- **TBL012**: تفاصيل السند (MainGuide → CardGuide للرأس)
- **TBL004**: دليل الحسابات
- **TBL001**: العملات
- **TBL005**: مراكز الكلفة
- **TBL049**: المشاريع
- **TBL050**: الفروع

## الخلاصة

يمكن إثراء عرض السند في الميل بإضافة:
1. **التاريخ** (InDate)
2. **الفرع** (من TBL050)
3. **مركز الكلفة** (من TBL005)
4. **المشروع** (من TBL049)
5. **العملة** (من TBL001)
6. **الحساب** (من TBL004 — للسندات ذات حساب واحد أو أول حساب)
7. **المجموع** (NmbCredit أو SUM من TBL012)

يُفضّل تنفيذ استعلام موسّع مع LEFT JOIN بحيث عند غياب أي جدول أو عمود لا يفشل الاستعلام كاملاً.
